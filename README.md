A 3d Software Renderer written in pure C.

#### Current Features
- Display 3d-models (.obj)
- Vertex-/Fragment-Saders
- Textures (PNG-Loader: http://lodev.org/lodepng/)
- Shadows
- Orbiting Camera
- Save to File
- Simple (Software only) Textrenderer
